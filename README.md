# Projektide portfoolio — Rahandus, Raamatupidamine ja SQL

See repositoorium koondab minu õpingute jooksul tehtud iseseisvaid praktilisi töid, mis demonstreerivad minu teadmisi ja praktilisi oskusi **finantsanalüüsi ja investeeringute hindamise**, **raamatupidamise** ning **andmebaaside disaini ja SQL-i** valdkonnas. Iga alamkaust sisaldab eraldi README-faili, kus on täpsemalt kirjeldatud kaustas olevate failide sisu.

Kõik failid on valminud õppetöö käigus iseseisvalt lahendatud harjutusülesannete ja praktikumide põhjal. Repositoorium sisaldab sealhulgas Excelis koostatud finantsmudeleid, raamatupidamisülesandeid, mis põhinevad kahekordsel kirjendamisel, ning SQL-päringuid ja relatsiooniliste andmebaaside disainilahendusi.

Linkidele vajutades saab iga valdkonnaga lähemalt tutvuda.

## Sisukord

| Valdkond | Mida sisaldab | Peamised oskused |
|---|---|---|
|  [**Rahandus**](./Rahandus-finantsanalüüs/) | Ettevõtte väärtuse hindamine (DCF, suhtarvud), kapitali hind (CAPM, WACC), investeeringute tasuvusanalüüs (NPV, IRR, MIRR), dividendipoliitika ja optsioonid | Finantsmodelleerimine Excelis, investeerimisotsuste analüüs |
|  [**Raamatupidamine**](./finantsarvestus-raamatupidamine/) | Kahekordne kirjendamine, T-kontod ja päevaraamat, korrigeerimiskanded, laovarude hindamine (FIFO), palga- ja puhkusetasuarvestus | Täieliku raamatupidamistsükli läbiviimine, tulude ja kulude vastavuse printsiip |
|  [**Andmebaasid**](./andmebaasid-SQL/) | Andmebaasi disain ja normaliseerimine, SQL päringud (JOIN, UNION, INTERSECT, NOT EXISTS), vaated, rekursiivsed seosed | Relatsiooniliste andmebaaside modelleerimine |

## Kasutatud tööriistad

- **Excel** — valemipõhised finants- ja raamatupidamismudelid
- **MS Access (.accdb)** — andmebaaside disain ja päringud 
- **MySQL (.sql)** — andmebaasi loomine ja päringud MySQL's
